# Corporate Branding Portfolio Website

This repository contains the design, planning, and frontend files for our agency's portfolio website. The site is designed to build customer trust and "relief" by showcasing high-fidelity interactive versions of our previous projects (LegalDiff and CareMate).

## Documentation & Planning Index

To guide our development and adhere to our custom design standards, we have divided the planning and specifications into the following documents:

1. **[PRODUCT.md](file:///D:/PROJECTS/Corporate_Branding/PRODUCT.md)**: Product description, goals, target audience segments, competitive inspiration, and section-by-section breakdown (Homepage, Projects, Q&A, and Contact).
2. **[DESIGN.md](file:///D:/PROJECTS/Corporate_Branding/DESIGN.md)**: Visual styling guidelines, typography scales, OKLCH design tokens, hover effects, motion principles, and design bans (preventing the typical AI slop aesthetic).
3. **[spec.md](file:///D:/PROJECTS/Corporate_Branding/spec.md)**: Spec-Driven Development technical specification detailing the tech stack, CLI commands, project structure, code style rules, and boundaries.
4. **[tasks/plan.md](file:///D:/PROJECTS/Corporate_Branding/tasks/plan.md)**: Technical execution plan detailing dependencies, risks, mitigations, and milestones.
5. **[tasks/todo.md](file:///D:/PROJECTS/Corporate_Branding/tasks/todo.md)**: Step-by-step checkable implementation tasks equipped with detailed acceptance and verification criteria.
